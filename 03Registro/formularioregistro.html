Habia una vez un patito que decia miau miau, y este patito queria aprender a utilizar diferentes tipos de formulas
Existen dos metodos principales para la obtencion de los datos dentro de un formulario web, para lo cual aqui lo explicaremos:


GET:

Este metodo es el estandar como salida del navegador mediante el cual se resuelve una peticion por medio del cliente, asi mismo el cual resuelve una peticion estandar(url), se puede visualizar los datos enviados mediante el formulario.

POST:

Este metodo es el mas utilizado para el envio de informacion desde un formulario atentiedno una peticion por parte del cliente actuando para poder agregar o insertar nuevos datos, asi mismo a diferencia del metodo get, este utiliza "un tipo de seguridad " mediante el cual la informacion no es visible por medio de la salida estandar del navegador.
